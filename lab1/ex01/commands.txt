1.
Название: tmux
Описание: Терминальный мультиплексор для работы с несколькими сессиями в одном терминале
Синтаксис: tmux [опции]
Пример: tmux
Примечание: Позволяет запускать процессы на сервере, отсоединяться и подключаться снова

2.
Название: rsync
Описание: Утилита для синхронизации и копирования файлов и каталогов между системами
Синтаксис: rsync [опции] источник назначение
Пример: rsync -avz /local/dir user@remote:/remote/dir
Примечание: Поддерживает инкрементную передачу и сжатие данных

3.
Название: fortune
Описание: Выводит случайные цитаты, шутки или мудрости
Синтаксис: fortune [опции]
Пример: fortune
Примечание: Часто используется вместе с cowsay для весёлого вывода

4.
Название: cowsay
Описание: ASCII-арт коровы (или других персонажей), говорящей заданный текст
Синтаксис: cowsay [опции] "текст"
Пример: fortune | cowsay

5.
Название: tree
Описание: Отображает содержимое каталогов в виде древовидной структуры
Синтаксис: tree [опции] [путь]
Пример: tree /etc
Примечание: Удобно для быстрого обзора структуры каталогов

6.
Название: ncdu
Описание: Интерактивный анализатор использования дискового пространства
Синтаксис: ncdu [опции] [путь]
Пример: ncdu /home
Примечание: Удобнее чем du благодаря интерфейсу с навигацией

7.
Название: grep
Описание: Поиск текста по шаблону в файлах или потоке
Синтаксис: grep [опции] "шаблон" [файлы]
Пример: grep "error" /var/log/syslog
Примечание: Поддерживает регулярные выражения

8.
Название: tldr
Описание: Выводит краткие, понятные примеры использования команд
Синтаксис: tldr [команда]
Пример: tldr rsync
Примечание: Отличный способ быстро вспомнить синтаксис

9.
Название: oneko
Описание: Забавная утилита, которая добавляет на экран маленького анимированного котёнка, бегущего за курсором мыши
Синтаксис: oneko [опции]
Пример: oneko
Примечание: Есть разные режимы (например,tora, sakura)

10.
Название: cmatrix
Описание: Утилита, отображающая анимацию в стиле «цифрового дождя» из фильма The Matrix
Синтаксис: cmatrix [опции]
Пример: cmatrix -u 0
Примечание: Есть разные режимы (быстрый, жирный шрифт, цветной), чисто визуальный эффект для терминала
